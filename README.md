#  Sales Performance Dashboard

## Overview
End-to-end sales data analysis project using Python.

## Steps Done
- Cleaned messy sales data using Pandas
- Built pivot tables by Region, Category, Segment
- Created interactive dashboard with 6 charts using Plotly

## Tools Used
- Python, Pandas, NumPy, Plotly, Jupyter Notebook

## Files
- sales_dashboard.html → Interactive dashboard
- sales_analysis.xlsx → Excel pivot tables
- cleaned_sales_data.csv → Cleaned dataset
